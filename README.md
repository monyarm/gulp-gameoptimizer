# Gulp-GameOptimizer

A gulpfile that is specifically made to optimize and reduce the size of games.

## Currently supported file formats:

### Executables

- EXE
- DLL
- PE
- LE
- ELF
- COM
- SYS
- SLUS/SLEU/SLJP

### Scripts and Markup:

- HTML (HTM)
- FTL
- CSS
- JS
- JSON
- Lua
- PY
- XML (XLF, SVG, RSS, XHTML, DAE)
- GLSL

### Resources:

- PNG
- JPG
- JPEG
- OBJ
